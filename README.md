## 🚀 Getting Started

1. Clone the repository

```bash
git clone https://github.com/Krishnasai-dev/portfolio.git
```

2. Open the project folder

```bash
cd portfolio
```

3. Open `index.html` in your browser.

No installation or dependencies are required.
